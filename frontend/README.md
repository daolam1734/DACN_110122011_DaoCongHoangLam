React demo app (Vite)

1. Cài đặt

```bash
cd frontend
npm install
```

2. Chạy dev server

```bash
npm run dev
```

3. Lưu ý
- App fetch dữ liệu từ `http://localhost:3001/vien-chuc` (backend phải đang chạy).
- Console sẽ log success/failure khi lấy dữ liệu (xem devtools hoặc terminal nếu server logs).
